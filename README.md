# OOPS Problems

This repository contains Java implementations demonstrating core Object-Oriented Programming (OOP) concepts through individual problem statements and a final application.

## Concepts Covered

- Encapsulation
- Inheritance
- Aggregation
- Association
- Method Overloading
- Composition
- Interfaces
- Polymorphism

## Implementations

- Encapsulation – Banking Problems
- Inheritance – Vehicle Logic
- Aggregation – University Problem
- Association – Employee & Department
- Method Overloading – Billing Utility
- Composition – House & Room
- Interfaces – Smart TV
- Polymorphism – Interface Implementation
- Food Delivery Application using OOP Concepts
